# 📈 Stock Market Analysis Dashboard  
> Comparative analysis of major tech stocks using data-driven insights and interactive visualizations

## 📖 Overview  
This project focuses on analyzing and comparing the stock market performance of four major technology companies—Apple, Microsoft, Netflix, and Google—using a three-month historical dataset (February to May 2023). The objective is to uncover trends, patterns, and relationships through a structured data analysis approach.

The analysis incorporates key financial indicators such as moving averages to identify trends, volatility to measure risk, and correlation analysis to evaluate how stocks move relative to each other. The results are presented through an interactive dashboard that enables users to explore stock behavior and market dynamics efficiently.

This project is designed for data analysts, finance enthusiasts, and decision-makers who want to gain quick, visual insights into stock performance and market sentiment within the tech sector.

## ✨ Features  
- 📊 Comparative analysis of multiple tech stocks (AAPL, MSFT, NFLX, GOOGL)  
- 📈 Trend visualization using line charts for stock prices  
- 🎯 Quick performance insights via gauge charts  
- 📉 Trading activity analysis using daily volume bar charts  
- 💧 Waterfall charts to break down net price changes over time  
- 🍩 Market sentiment visualization (bullish vs bearish days)  
- 🔗 Correlation analysis to understand stock co-movement  
- ⚡ Identification of volatility and risk patterns  

## 🛠️ Tech Stack  
- Power BI (dashboard creation and visualization)  
- Data analysis techniques:
  - Moving averages  
  - Volatility analysis  
  - Correlation analysis  

## 📊 How It Works  
The project follows a structured analytical workflow:

1. **Data Collection**  
   - Historical stock data for four companies over a three-month period  

2. **Data Processing & Analysis**  
   - Moving averages applied to identify trends  
   - Volatility calculated to assess risk levels  
   - Correlation analysis performed to study relationships between stocks  

3. **Visualization Layer (Power BI Dashboard)**  
   - **Line Chart:** Displays stock price trends over time  
   - **Gauge Chart:** Indicates current price relative to yearly highs  
   - **Bar Chart (Volume):** Highlights trading activity and spikes  
   - **Waterfall Chart:** Breaks down gains and losses across periods  
   - **Donut Chart:** Shows market sentiment (bullish vs bearish days)  

4. **Insights Generation**  
   - Market recovery observed in mid-March with upward momentum into April  
   - Sector divergence identified across different stocks  
   - Strong correlation between price movements and trading volume spikes  
   - Predominantly bullish sentiment across the selected period
